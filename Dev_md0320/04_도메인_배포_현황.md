## 2. 배포 방법

### 로컬 빌드 배포 (주력)
```bash
cd D:/dreamit-web/{project}
npm run build
npx gh-pages -d dist
```

### Next.js 프로젝트 배포
```bash
cd D:/dreamit-web/{project}
npm run build    # next build && next export
npx gh-pages -d out
```

### GitHub Actions (일부)
- openclaw: `.github/workflows/deploy.yml` 존재
- 대부분 로컬 빌드 후 gh-pages 명령으로 배포

## 4. SPA 라우팅 처리

모든 React SPA 프로젝트는 `public/404.html`에 리다이렉트 스크립트 포함:
- GitHub Pages는 SPA 라우팅 미지원
- 404.html에서 URL을 쿼리파라미터로 변환 후 index.html로 리다이렉트
- index.html에서 쿼리파라미터를 원래 경로로 복원
