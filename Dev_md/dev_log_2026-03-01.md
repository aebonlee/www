## 세션 2: 출판 도서 페이지 book-card 디자인 개선

### 작업 배경

`/publishing/book` 페이지의 "출판 도서" 섹션에서 도서 카드 내부 간격이 좁고 시각적 임팩트가 부족하여 디자인 재검토 요청.

### 문제점 분석

1. 카테고리 뱃지와 가격 구분선 사이 여백 없음 (답답)
2. 도서 커버 80x110px — 실제 책 비율보다 작음
3. 제목 폰트 16px — 제목으로서 존재감 부족
4. 카드 전체적으로 빽빽한 레이아웃
5. hover 효과 약함

### 수정 파일

| 파일 | 변경 사항 |
|------|----------|
| `src/styles/education.css` | book-card 레이아웃 전면 개선 |
| `src/styles/shop.css` | book-purchase 간격 확대, 가격 폰트 키움 |
