## 세션 2: visited_sites 자동 등록

### 작업 배경

기존에는 관리자가 수동으로 회원의 `visited_sites`를 편집해야 했음. 회원이 하위 사이트에 로그인(또는 세션 복원)하면 현재 도메인이 자동으로 `visited_sites` 배열에 추가되도록 개선.

### 구현 로직

`AuthContext.jsx`의 `loadProfile` 함수에서:
1. 현재 도메인(`window.location.hostname`) 확인
2. `user_profiles.visited_sites` 배열에 현재 도메인이 없으면 자동 추가
3. `signup_domain`이 비어있으면 현재 도메인으로 초기화
4. `role`이 `'user'`이면 `'member'`로 자동 변경

### 변경 파일

| 파일 | 변경 내용 |
|------|----------|
| `contexts/AuthContext.jsx` (본사이트) | `loadProfile`에 visited_sites 자동 추가 로직 |
| `contexts/AuthContext.jsx` (템플릿) | 동일 변경 반영 |
