### 3.2 회사소개/연혁 페이지 분리

**커밋**: `42552e0` feat: 회사소개/연혁 페이지 분리 — 해시 링크 제거, History 독립 페이지 신설

**문제**: 해시 링크(`/about#company`, `/about#vision`, `/about#history`)가 HashRouter 환경에서 앵커 이동이 불안정

**해결**: 페이지 분리

| 변경 전 | 변경 후 |
|---------|---------|
| `/about#company` (해시 앵커) | `/about` (독립 페이지 — 회사 개요 + 비전) |
| `/about#vision` (해시 앵커) | `/about` (같은 페이지에 포함) |
| `/about#history` (해시 앵커) | `/about/history` (독립 페이지) |

**수정 파일:**
- `About.jsx` — 연혁 섹션 제거 (회사 개요 + 비전 & 미션 + 핵심 가치만 유지)
- `History.jsx` — **신규 생성**. 연혁 전용 페이지, 1차 창업기 / 2차 창업기 구간 분리, 한/영 다국어 지원
- `App.jsx` — `/about/history` 라우트 추가
- `Navbar.jsx` — 해시 링크 제거, 독립 경로로 변경
- `translations.js` — `about.company` + `about.vision` → `about.companyVision` 통합

**회사소개 메뉴 최종 구조:**
```
회사소개
├── 회사소개 → /about (회사 개요 + 비전 & 미션 + 핵심 가치)
├── 연혁 → /about/history (1차 창업기 / 2차 창업기 구간 분리)
├── 대표 소개 → /about/ceo
└── 연락처 → /contact
```

### 3.3 Home.jsx 번역 키 하드코딩 수정

**이전:**
```jsx
<h2 className="section-title">Our Services</h2>
<h2 className="section-title">Portfolio</h2>
<h2 className="section-title">고객 후기</h2>
```

**수정 후:**
```jsx
<h2 className="section-title">{t('sections.ourServices')}</h2>
<h2 className="section-title">{t('sections.portfolio')}</h2>
<h2 className="section-title">{t('sections.reviews')}</h2>
```

- 영문 모드(EN) 전환 시에도 정상 표시됨
- 경력 통계도 `5+` → `22+`로 수정 (About 페이지와 일치)

```jsx
{ path: '/services/web-development', ..., tKey: 'webDev' }
// 렌더링:
<h3>{t(`homeServices.${service.tKey}.title`)}</h3>
<p>{t(`homeServices.${service.tKey}.desc`)}</p>
{[1, 2, 3].map((n) => <li key={n}>{t(`homeServices.${service.tKey}.f${n}`)}</li>)}
```
