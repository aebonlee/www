## 1. 작업 개요

기존 대시보드는 11개 stat 카드가 그루핑 없이 1차원 나열되고, 최근 주문/콘텐츠 2개 섹션만 있는 단순 구조였다. Shopify, WooCommerce 스타일의 전문적 관리자 패널로 전면 리디자인했다.

**Before**: 11개 카드 나열 + 2개 리스트 섹션
**After**: 6개 Zone 기반 정보 계층화 레이아웃

## 3. 새 대시보드 레이아웃 (6개 Zone)

```
┌──────────────────────────────────────────────────┐
│ ZONE 1: Welcome Header (그라디언트 배너)           │
│ "안녕하세요, 관리자님" | 날짜 | 요약 | 새로고침      │
├──────────────────────────────────────────────────┤
│ ZONE 2: Commerce Hero Stats (4개 대형 카드)        │
│ [총 매출] [결제완료] [총 주문] [취소/환불]           │

│ ZONE 3a: 콘텐츠 & 회원  │ ZONE 3b: 매출 현황       │
│ 6개 소형 stat 카드       │ CSS 비율 바 + 3개 지표    │
│ (2열 그리드)            │ 결제성공률, 평균주문금액    │

│ ZONE 4a: 최근 주문      │ ZONE 4b: 콘텐츠 분포      │

│ 활동 피드 (아이콘 포함)  │ 6개 퀵 액션 버튼           │
└────────────────────────┴─────────────────────────┘
```

## 4. 구현 상세

### 4.1 Zone 1: Welcome Header
- `useAuth()`에서 `profile.full_name` / `user.email` 가져옴
- `var(--primary-gradient)` 배경 → 다크모드 자동 대응
- `toLocaleDateString('ko-KR', ...)` 한국어 날짜 포맷
- 요약 텍스트: "주문 N건 · 결제완료 N건 · 회원 N명"
- 새로고침 버튼을 배너 우측으로 이동 (기존 `admin-page-header` 제거)

### 4.2 Zone 2: Commerce Hero Stats (4개 대형 카드)
- 총 매출(`formatPrice`), 결제완료, 총 주문, 취소/환불
- 기존 `AdminStatCard` 100% 재사용, `.dash-hero-stats` 래퍼로 크기 확대
- 4열 → 2열(태블릿) → 1열(모바일) 반응형

### 4.3 Zone 3a: 콘텐츠 & 회원 (6개 소형 카드)
- 회원, 블로그, 갤러리, 게시판, 강의계획서, 상품
- `.dash-compact-stats` 래퍼로 컴팩트 사이즈 (아이콘 36px, 값 20px)
- 2열 그리드

### 4.4 Zone 3b: 매출 현황
- 결제/취소 금액 비율을 CSS `linear-gradient` 프로그레스 바로 시각화
- 하단 3개 지표 (grid 3열): 총 거래 건수, 결제 성공률(%), 평균 주문 금액
- 순수 CSS — 차트 라이브러리 의존 없음

### 4.5 Zone 4a: 최근 주문 (개선)
- 금액 `formatPrice()` + 날짜 `formatDate()` 별도 표시
- 기존 status badge 시스템 유지
- purple 뱃지 신규 추가 (환불 상태용)

### 4.6 Zone 4b: 콘텐츠 분포
- 5개 콘텐츠 유형(블로그/갤러리/게시판/강의계획서/상품) 비율을 CSS 스택 바로 시각화
- 색상 범례 + 건수 표시
- `getGalleryItems()` 추가 호출로 정확한 갤러리 카운트 확보

### 4.7 Zone 5a: 최근 콘텐츠
- 기존 블로그+게시판 머지 로직 유지
- 타입별 아이콘 추가 (`.dash-type-icon`)

### 4.8 Zone 5b: 바로가기
- 6개 주요 관리 페이지 (상품/주문/블로그/회원/갤러리/게시판)
- 2열 그리드, 아이콘 + 라벨, hover 효과

## 5. CSS 설계

### 신규 CSS 클래스 체계 (`dash-*` prefix)
| 구분 | 주요 클래스 |
|------|------------|
| 컨테이너 | `.dash-container` |
| 배너 | `.dash-welcome`, `.dash-welcome-content`, `.dash-refresh-btn` |
| 히어로 | `.dash-hero-stats` |
| 분할 행 | `.dash-split-row` |
| 패널 | `.dash-panel`, `.dash-panel-header`, `.dash-panel-body` |
| 컴팩트 | `.dash-compact-stats` |
| 매출 바 | `.dash-revenue`, `.dash-bar-*`, `.dash-metric` |
| 콘텐츠 분포 | `.dash-stack-bar`, `.dash-stack-segment`, `.dash-legend-*` |
| 바로가기 | `.dash-quick-actions`, `.dash-quick-btn`, `.dash-quick-icon` |
| 뱃지 | `.td-badge.purple` (신규) |

### 반응형 브레이크포인트
| 해상도 | 동작 |
|--------|------|
| > 1024px | 풀 레이아웃 (4열 히어로, 2열 분할) |
| 768~1024px | 히어로 2x2, 분할 행 스택 (1열) |
| 480~768px | 히어로 2열, 컴팩트 2열, 퀵액션 2열 |
| < 480px | 모든 그리드 1열 |
