## 1. 작업 개요

3차 이커머스 개발이 완료된 상태에서 두 가지 작업을 수행했다:

1. **연락처(Contact) 메뉴를 회사소개(About) 드롭다운 하위로 이동**
2. **CEO 프로필(대표 소개) 페이지 신규 생성 및 회사소개 메뉴에 추가**

CEO 이애본 대표의 경력, 전문분야, 학력, 자격증, 수상 경력, 연락처를 포함하는 상세 프로필 페이지를 한/영 이중언어로 구현했다.

## 6. 설계 판단

**1) 전용 CSS 파일 분리**

CEO 프로필은 타임라인, 전문분야 카드, 자격증 그리드 등 다른 페이지에서 재사용하지 않는 고유 레이아웃이 많다. 기존 CSS에 추가하면 파일이 비대해지므로 `ceo.css`로 분리했다.

**2) 번역 파일 대신 컴포넌트 내 조건부 렌더링**

CEO 프로필의 텍스트는 경력, 학력 등 고유 콘텐츠로서 재사용되지 않는다. 이를 모두 `translations.js`에 넣으면 파일이 과도하게 비대해진다. 대신 `language === 'ko' ? '한글' : 'English'` 패턴으로 컴포넌트 내에서 직접 처리했다. 메뉴명(`about.ceoProfile`, `about.contact`)만 translations.js에 추가했다.

**3) SVG 아바타 사용**

실제 CEO 사진이 없으므로 User SVG 아이콘 + 파란 원형 배경으로 아바타를 구현했다. 추후 사진이 준비되면 `<img>` 태그로 교체하면 된다.

**4) 기존 /contact 라우트 유지**

연락처 메뉴를 회사소개 하위로 이동했지만, `/contact` URL 라우트는 그대로 유지했다. 이미 검색엔진이나 외부 링크에서 `/contact`로 접근하는 경우가 있을 수 있기 때문이다. 메뉴 위치만 변경하고 라우트는 건드리지 않았다.
