# 6차 개발: 가독성 개선 + 컨설팅 메뉴 신설

**작성일**: 2026-02-18
**담당**: Claude Opus 4.6
**프로젝트**: DreamIT Biz React SPA
**상태**: 완료

---

## 개요

5차 디자인 리뉴얼 이후, 전반적인 가독성을 재검토하여 폰트 크기·색상 대비·라인하이트를 최적화하고, 푸터 메뉴 레이아웃을 개선하였다. 또한 IT서비스 하위에 있던 "기업 컨설팅"을 독립 상위 메뉴 "컨설팅"으로 승격시키고, 기업 컨설팅·대학 컨설팅·교육기관 컨설팅 3개 하위 페이지를 신규 생성하였다.

## B. 컨설팅 상위 메뉴 신설

### 1. 메뉴 구조 변경

**Before:**
```
IT서비스
├── 웹개발
├── 웹호스팅
├── 디자인
└── 기업컨설팅 ← 여기에 있었음
```

**After:**
```
IT서비스
├── 웹개발
├── 웹호스팅
└── 디자인

컨설팅 (신규 상위 메뉴)
├── 기업 컨설팅
├── 대학 컨설팅
└── 교육기관 컨설팅
```

### 4. 라우트 구조

```
/consulting                → Consulting.jsx (카테고리 선택)
/consulting/business       → ConsultingBusiness.jsx
/consulting/university     → ConsultingUniversity.jsx
/consulting/institution    → ConsultingInstitution.jsx
```

### 5. 각 하위 페이지 구성

Education/Publishing 패턴과 동일한 구성:
- **Page Header**: 블루 그라데이션 배경 + 타이틀/서브타이틀
- **Overview**: 2열 그리드 (텍스트 + SVG 일러스트)
- **Features**: 3열 feature-card 그리드 (6개 항목)
- **Process**: 4단계 프로세스 스텝
- **CTA**: 그라데이션 배경 + 문의 버튼

### 6. 컨설팅 상세 데이터

**기업 컨설팅**: IT 전략 수립, 프로세스 개선, 기술 도입 지원, 보안 컨설팅, 데이터 분석, 변화 관리
**대학 컨설팅**: 스마트 캠퍼스, LMS 구축, 비대면 교육, 학사 시스템, 데이터 기반 교육, 산학 협력
**교육기관 컨설팅**: 스마트 교실, 교육 콘텐츠, 행정 시스템, AI 교육 도구, 정보보안, 교사 연수
