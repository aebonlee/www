# 5차 개발: 디자인 리뉴얼 (Pfizer + Weniv 참고)

**작성일**: 2026-02-18
**담당**: Claude Opus 4.6
**프로젝트**: DreamIT Biz React SPA
**상태**: 완료

---

## 개요

DreamIT Biz React SPA의 기능 구현(1~4차)이 완료된 상태에서, Pfizer(깊은 블루, 글래스모피즘, 넉넉한 여백)와 Weniv.co.kr(한국 IT교육 기업, 프로페셔널)을 참고하여 더 프리미엄하고 모던한 느낌으로 CSS를 전면 개선하였다.

### 핵심 원칙
- CSS 파일만 수정 (JSX 구조 변경 없음)
- 다크모드 호환 유지
- 반응형 유지
- 기존 클래스명 그대로 사용

---

### 1. 디자인 토큰 (`base.css`)

| 속성 | Before | After |
|------|--------|-------|
| Primary Blue | `#0066CC` | `#0046C8` (깊은 블루) |
| Primary Blue Dark | `#004C99` | `#002E8A` |
| Primary Blue Light | `#3385D6` | `#4A8FE7` |
| BG Light Gray | `#F7F9FC` | `#F5F7FA` |
| Text Primary | `#1A1A1A` | `#111827` (더 깊은 차콜) |
| Text Secondary | `#4A4A4A` | `#4B5563` |
| Text Light | `#6B6B6B` | `#6B7280` |

| Shadow MD | 단일 레이어 | 이중 레이어 |

#### 신규 CSS 변수
- `--primary-gradient`: 블루 그라데이션
- `--accent-gradient`: 3-stop 블루 그라데이션
- `--glass-bg`, `--glass-border`, `--glass-blur`: 글래스모피즘
- `--radius-sm(8px)`, `--radius-md(12px)`, `--radius-lg(16px)`, `--radius-xl(20px)`, `--radius-full(9999px)`
- `--shadow-xl`: 최대 그림자
- `--transition-base`: cubic-bezier(0.4, 0, 0.2, 1) 이징

### 2. 글래스모피즘 네비바 (`navbar.css`)
- 배경: `rgba(255,255,255,0.85)` + `backdrop-filter: blur(12px)`
- 스크롤 시: `rgba(255,255,255,0.95)`, border 투명
- 드롭다운: `border-radius 12px`, `backdrop-filter blur(16px)`, 반투명 배경
- 언더라인: `border-radius 2px` 추가

### 3. 히어로 섹션 (`hero.css`)
- 높이: `600px` → `min(85vh, 800px)`
- 배경: 3-stop 블루 그라데이션 (`#0046C8` → `#002E8A` → `#1E3A5F`)
- 타이틀: `56px` → `60px`, `letter-spacing -0.03em`
- `.highlight`: 그라데이션 텍스트 효과 (`background-clip: text`)
- 파티클: 크기 `4px` → `6px`, 흰색 + 더 은은하게
- 스크롤 인디케이터: 색상 연하게 (반투명 흰색)

### 4. 서비스 카드 (`services.css`)
- 카드: `border-radius 16px`, 호버 시 `translateY(-6px)` + `shadow-xl`
- 아이콘: gradient 배경 + `box-shadow`
- 프로세스 넘버: gradient 배경
- 가격 카드: `radius 16px`, featured `2px` 보더

### 5. 포트폴리오 (`portfolio.css`)
- 페이지 헤더: 3-stop 블루 그라데이션 (흰색 텍스트)
- 카드: `radius 16px`, 호버 `shadow-xl`
- 태그: pill 형태 (`9999px`)
- 프로젝트 뱃지: gradient 배경, pill 형태

### 6. 회사소개 (`about.css`)
- 통계 숫자: gradient 텍스트 (`background-clip: text`)
- 가치 카드: `radius 16px`, 호버 `translateY(-4px)`
- 비전 카드: `radius 16px`, 제목 gradient 텍스트
- 타임라인 선: gradient + 투명도

### 7. CTA + 리뷰 (`cta.css`)
- CTA: gradient 배경 + 장식 원형 `pseudo-element` (::before, ::after)
- 리뷰 카드: `radius 16px`
- 리뷰 텍스트: 좌측 gradient 인용 라인 (`border-image`)

### 8. 푸터 (`footer.css`)
- 배경: 단색 → gradient (`#111827` → `#0A0F1A`)
- 링크 호버: `#93C5FD` (더 밝은 블루)
- 하단 구분선: `rgba(255,255,255,0.06)` (더 은은하게)

### 9. 블로그 (`blog.css`)
- 카드: `radius 16px`, 호버 `shadow-xl`
- 카테고리 뱃지: pill 형태, `rgba(0,70,200,0.06)` 배경

### 10. 연락처 (`contact.css`)
- 폼: `radius 16px`, 보더 추가
- 인풋 포커스: 블루 ring (`box-shadow: 0 0 0 3px rgba(0,70,200,0.1)`)
- 아이콘 박스: gradient 배경 + box-shadow

### 11. 애니메이션 (`animations.css`)
- 전체: `0.6s ease` → `0.7s cubic-bezier(0.4, 0, 0.2, 1)`
- 이동 거리: `30px` → `24px` (더 정제됨)
- `zoom-in`: `0.9` → `0.95` (더 섬세)

### 12. 교육 페이지 (`education.css`)
- 카드류: `radius 8px` → `12px` (var(--radius-md))
- 카테고리 카드: `radius 16px`
- 필터 버튼/메타 뱃지: pill 형태 (9999px)
- 아이콘: gradient 배경 + box-shadow

### 13. 쇼핑 (`shop.css`)
- 상품 카드: `radius 16px`
- 장바구니/결제 요약: `radius 12px`
- 결제 옵션: `radius 12px`

### 14. CEO 프로필 (`ceo.css`)
- 프로필 카드/연락처 박스: `radius 16px`
- 전문분야/자격증 카드: `radius 12px`
- 아이콘: gradient 배경 + box-shadow
- 타임라인 선: gradient + 투명도

### 15. 다크모드 (`dark-mode.css`)
- Tailwind Gray 팔레트 기반 (쿨한 톤):
  - `bg-white`: `#1A1A1A` → `#111827`
  - `bg-light-gray`: `#2A2A2A` → `#1F2937`
  - `bg-medium-gray`: `#3A3A3A` → `#374151`
  - `text-secondary`: `#B0B0B0` → `#D1D5DB`
  - `text-light`: `#888888` → `#9CA3AF`
- 히어로/페이지헤더: 인디고 틴트 다크 그라데이션
- 푸터: gradient (`#0F172A` → `#030712`)
- 네비바: 다크 글래스모피즘 (`rgba(17,24,39,0.85)` + blur)
- CTA: 다크 블루+인디고 그라데이션
- 드롭다운: 다크 글래스모피즘

### 16. 반응형 (`responsive.css`)
- 모바일 드롭다운: `radius 8px`
- 히어로 타이틀 크기 미세 조정 (`46px` → `34px` → `28px`)
- 모바일 버튼: `radius 12px` 유지
- 모바일 드롭다운: backdrop-filter 비활성화 (성능)
