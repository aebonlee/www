## 5. 기능 상세

### 5.1 다크/라이트 모드
- localStorage 기반 테마 저장
- CSS custom properties 활용
- 부드러운 전환 애니메이션

### 5.2 한국어/영어 전환
- data-i18n 속성 기반 번역 시스템
- 주요 섹션 제목, 본문, 버튼 텍스트 영어 번역 포함

### 5.3 스크롤 애니메이션
- Intersection Observer 기반 fade-up 애니메이션
- 배지(badge) 순차 등장 효과 (60ms 간격)
- 히어로 섹션 패럴랙스 효과

### 5.4 카운터 애니메이션
- 숫자가 0에서 목표치까지 2초간 카운트업
- easeOutCubic 이징 함수 적용

### 5.5 반응형 디자인
- Desktop (1200px+): 3열 그리드
- Tablet (768–1024px): 2열/1열 적응
- Mobile (768px 이하): 전체 1열, 햄버거 메뉴
