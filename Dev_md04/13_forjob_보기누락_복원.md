## 문제 원인

### PDF 텍스트 추출 한계

comcbt.com에서 제공하는 직업상담사 2급 교사용 PDF 파일(55개)에서 `pdf-parse` 라이브러리로 텍스트를 추출할 때, **보기(ㄱ.ㄴ.ㄷ.) 블록**과 **지문/표/그래프**가 이미지(벡터 그래픽)로 렌더링되어 텍스트 레이어에 포함되지 않음.

검증 도구:
- `pdf-parse` (PDFParse class): getText() → 보기 텍스트 없음
- `pdfjs-dist` (직접 API): 텍스트 아이템 추출 → 보기 없음 (73개 한글 자음만 옵션에서 발견)
- 상세 위치 분석: Y/X 좌표별 아이템 확인 → 보기 영역에 텍스트 객체 없음

### 누락 유형 분류

| 유형 | 설명 | 예시 |
|------|------|------|
| 보기 블록 | ㄱ.ㄴ.ㄷ. 정의가 누락 | "모두 고른 것은?" 유형 |
| 지문/사례 | "다음 설명에 해당하는..." 뒤 본문 누락 | 상담 대화, 법조문, 이론 설명 |
| 표 | 비교표, 통계표 등 | 직업정보 유형별 표, 인구통계 |
| 그래프 | 노동경제학 그래프 설명 | 임금-노동공급 곡선 |
| 순서 나열 | 단계별 순서 보기 | Williamson 6단계 등 |

## 복원 과정

### 1단계: 문제 탐지 스크립트

```javascript
// scripts/find-broken.cjs
const problematic = data.filter(q => {
  const opts = [q.option_1, q.option_2, q.option_3, q.option_4].join(' ');
  const hasRef = /[ㄱㄴㄷㄹㅁㅂ]/.test(opts);
  const textHasRef = /[ㄱㄴㄷㄹㅁㅂ]/.test(q.question_text || '');
  return hasRef && !textHasRef;
});
```

초기 탐지: 105개 (ㄱㄴㄷ 참조 문제만)
실제 복원: ~280개 (지문/표/그래프 누락 포함)

### 2단계: 자동 복원 시도 (실패)

- PDF 재파싱: 보기 블록이 이미지이므로 불가
- 웹 검색: CBT 사이트가 JavaScript 동적 렌더링이라 크롤링 불가
- 도메인 지식 역산: 순서 나열 문제 일부만 가능 (fix-with-knowledge.cjs)

### 3단계: 수동 복원 (성공)

55개 PDF 파일을 수동으로 확인하여 누락 텍스트를 연도/회차별로 제공받아 JSON에 추가.
