## 5. 코드 패턴 표준

### OAuth 로그인
```javascript
const { error } = await supabase.auth.signInWithOAuth({
  provider: 'google', // 또는 'kakao'
  options: { redirectTo: window.location.origin }
});
```

### signUp 후 프로필 생성
```javascript
const { data, error } = await supabase.auth.signUp({
  email,
  password,
  options: { data: { full_name: name } }
});
if (data.user) {
  await supabase.from('user_profiles').upsert({
    id: data.user.id,
    email: data.user.email,
    full_name: name,
    signup_domain: window.location.hostname,
    visited_sites: [window.location.hostname]
  }, { onConflict: 'id' });
}
```

### 도메인 추적 (onAuthStateChange)
```javascript
supabase.auth.onAuthStateChange(async (event, session) => {
  if (event === 'SIGNED_IN' && session?.user) {
    const hostname = window.location.hostname;
    const { data: profile } = await supabase
      .from('user_profiles')
      .select('visited_sites')
      .eq('id', session.user.id)
      .single();
    const sites = profile?.visited_sites || [];
    if (!sites.includes(hostname)) {
      await supabase.from('user_profiles').update({
        visited_sites: [...sites, hostname],
        last_sign_in_at: new Date().toISOString()
      }).eq('id', session.user.id);
    }
  }
});
```

### 비밀번호 정책
```javascript
const passwordRegex = /^(?=.*[a-zA-Z])(?=.*\d).{8,}$/;
if (!passwordRegex.test(password)) {
  setError('비밀번호는 8자 이상, 영문과 숫자를 포함해야 합니다.');
  return;
}
```
