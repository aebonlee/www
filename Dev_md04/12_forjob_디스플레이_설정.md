## 주요 변경

### 1. 디스플레이 설정 툴바

문항 영역 상단에 보기 모드 + 크기 조절 토글 버튼 배치.
- 1열/2열 레이아웃 전환 (1280px+ 데스크탑 전용)
- 작게/보통/크게 문항 크기 변경

### 2. 2열 보기 모드

- CSS Grid `1fr 1fr` 레이아웃
- 2문항씩 페이지 이동, 페이지 표시 형식: `1–2 / 100`
- `exam-body.dual-mode`에서 `max-width: none` 적용
- 마지막 페이지 홀수 문항 시 왼쪽만 표시

### 3. 크기 조절

| 크기 | question-text | option-text | card padding |
|------|---------------|-------------|--------------|
| 작게 | 14px | 13px | 24px |
| 보통 | 16px | 15px | 32px |
| 크게 | 18px | 17px | 40px |

### 4. 반응형 처리

- 1279px 이하: 2열 토글 숨김
- 767px 이하: 툴바 축소
- 639px 이하: 라벨 숨김, 중앙 정렬
