# ForJob - 다크모드 보기/표/지문 스타일 수정

> 작성일: 2026-04-06
> 프로젝트: ForJob (직업상담사 2급 시험 준비 플랫폼)
> 배포: https://forjob.dreamitbiz.com

---

## 작업 요약

| 항목 | 내용 |
|------|------|
| 문제 | 다크모드에서 보기/표/지문 블록이 흰색 배경 + 흰색 글씨로 보이지 않음 |
| 원인 | 14번 작업에서 추가한 `.bogi-box`, `.passage-block`, `.q-table` 등의 CSS가 하드코딩된 라이트 색상 사용 |
| 해결 | `dark-mode.css`에 `html[data-theme="dark"]` 셀렉터로 다크 테마 색상 오버라이드 추가 |
| 영향 범위 | 보기(ㄱ.ㄴ.ㄷ.) 블록, 지문 블록, 표 — 다크모드에서 정상 표시 |

---

## 문제 상세

### 증상
- 다크모드 전환 시 보기 박스 배경이 `#F8FAFC`(거의 흰색), 텍스트도 밝은 색 → 글씨 안 보임
- 지문 박스 배경 `#FFFBEB`(연한 노란색), 다크 배경과 충돌
- 표 셀 배경 `#FFFFFF`, 헤더 배경 `#E8EEF6` → 다크 배경에서 눈부심

### 원인
14번 작업(보기/표/지문 시각화 렌더링)에서 `exam.css`에 추가한 스타일이 고정된 라이트 색상을 사용했으나, `dark-mode.css`에 대응하는 다크 오버라이드를 추가하지 않음.

## 색상 설계 원칙

| 요소 | 라이트 모드 | 다크 모드 | 의도 |
|------|-----------|----------|------|
| 보기 배경 | 연한 회색 (`#F8FAFC`) | 진한 슬레이트 (`#1E293B`) | 주변 카드와 구분 |
| 보기 라벨 | 파란 뱃지 | 어두운 뱃지 + 밝은 파란 텍스트 | 라벨 가독성 |
| 지문 배경 | 연한 노란색 (`#FFFBEB`) | 어두운 갈색 (`#1C1917`) | 경고/주목 톤 유지 |
| 지문 라벨 | 갈색 (`#92400E`) | 노란색 (`#FCD34D`) | 밝은 배경↔어두운 배경 반전 |
| 표 헤더 | 연한 파란 배경 | 진한 파란 배경 (`#1E3A5F`) | 헤더-데이터 구분 유지 |
| 표 셀 | 흰색 | 거의 검정 (`#0F172A`) | 최대 대비 |
