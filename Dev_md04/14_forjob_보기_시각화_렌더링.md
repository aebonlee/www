## 렌더링 규칙

### 1. 보기 블록 (`<보기>`)

**탐지 패턴**: `\n\n<보기>\n` 으로 분리

**렌더링**:
- 파란 테두리 박스 (`#CBD5E1`)
- "보기" 라벨 (파란 배지)
- ㄱ.ㄴ.ㄷ. 패턴 → 리스트 형태 (하단 구분선)

```
┌──────────────────────────────┐
│ [보기]                        │
│ ㄱ. 분석                      │
│ ────────────────────────────  │
│ ㄴ. 종합                      │
│ ────────────────────────────  │
│ ㄷ. 진단                      │
└──────────────────────────────┘
```

### 2. 표 (Markdown 테이블)

**탐지 패턴**: `|`로 시작하고 끝나는 줄이 2줄 이상

**렌더링**:
- 파란 박스 내부 HTML `<table>`
- 첫 행: 헤더 (파란 배경 `#E8EEF6`, 볼드)
- 나머지: 흰 배경 데이터 셀
- 구분선(`|---|`) 자동 건너뜀

```
┌──────────────────────────────┐
│ 구분   │ 비용 │ 접근성       │
│────────┼──────┼─────────────│
│ 인쇄물 │ 저   │ 용이         │
│ 면접   │ 저   │ 제한적       │
└──────────────────────────────┘
```

### 3. 지문 블록

**탐지 패턴**: `\n\n다음 [설명/사례/대화/조건/표/그래프/공식]에 해당하는...\n\n`

**렌더링**:
- 노란 테두리 박스 (`#F59E0B`)
- 질문부 라벨 (갈색 텍스트)
- 지문 본문

```
┌──────────────────────────────┐
│ 다음 설명에 해당하는 것은?     │
│                              │
│ 내담자에게 과거에 했던 선택의   │
│ 회상, 절정경험, 자유시간...    │
└──────────────────────────────┘
```

### 4. 일반 텍스트

위 패턴에 매칭되지 않으면 `white-space: pre-wrap`으로 줄바꿈 유지.

## 수정 파일

### `src/components/QuestionCard.tsx`

| 함수 | 역할 |
|------|------|
| `renderQuestionText()` | question_text를 파싱하여 블록 분리 |
| `renderInnerContent()` | ㄱ.ㄴ.ㄷ. 패턴 감지 → 리스트 렌더링 |
| `renderTextBlock()` | 일반 텍스트 블록 렌더링 |

파싱 우선순위:
1. `<보기>` 블록 분리
2. Markdown 테이블 감지
3. 지문 블록 감지
4. 일반 텍스트 fallback

### `src/styles/exam.css`

| 클래스 | 설명 |
|--------|------|
| `.bogi-box` | 보기 박스 (파란 테두리, 둥근 모서리) |
| `.bogi-label` | "보기" 라벨 배지 |
| `.bogi-content` | 보기 내용 영역 |
| `.bogi-list` | ㄱ.ㄴ.ㄷ. 리스트 |
| `.bogi-list li` | 리스트 아이템 (하단 구분선) |
| `.passage-block` | 지문 박스 (노란 테두리) |
| `.passage-label` | 지문 질문부 라벨 |
| `.passage-content` | 지문 본문 영역 |
| `.q-table` | 보기 내 테이블 |
| `.q-table-header th` | 테이블 헤더 (파란 배경) |

### 크기별 반응형

| 크기 | 박스 패딩 | 내용 폰트 | 테이블 폰트 |
|------|----------|----------|------------|
| 작게 (size-sm) | 12px 14px | 13px | 12px |
| 보통 (기본) | 16px 20px | 15px | 14px |
| 크게 (size-lg) | 20px 24px | 17px | 16px |
