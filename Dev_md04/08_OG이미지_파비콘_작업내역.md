## 3. 파비콘 생성 (7개 프로젝트)

### 도구
- Node.js 22 + sharp 0.34.5
- SVG → PNG/ICO 변환
- 스크립트: `D:\dreamit-web\generate-favicons.js`

### 생성 파일 (프로젝트당 4개)
| 파일 | 크기 | 용도 |
|------|------|------|
| `favicon.svg` | 벡터 | 브라우저 우선 표시 |
| `favicon.ico` | 48×48 | 레거시 브라우저 호환 |
| `favicon-32x32.png` | 32×32 | 표준 파비콘 |
| `favicon-192x192.png` | 192×192 | PWA / apple-touch-icon |
