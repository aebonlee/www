## 1. 필기 기출문제 PDF 추출

### 추출 범위

| 연도 | 회차 | 문항수 |
|------|------|--------|
| 2010 | 1~4회 | 400 |
| 2011~2015 | 각 1~3회 | 1,500 |
| 2016~2019 | 각 1~3회 | 1,200 |
| 2020~2021 | 각 1~3회 | 600 |
| 2022 | 1~2회 | 200 |
| **합계** | **39회** | **3,900** |

- 빈 보기: 0개, 정답 누락: 0개
- 해설 포함: 721/3,900 (8개 회차분)

### 기술 구현

- `pdf-parse` 라이브러리 (`PDFParse` 클래스) 사용
- 정답 마커: `❶❷❸❹` (유니코드 U+2776~2779)
- 보기 분리: `①②③④` 분할 + 정답표 영역 제거
- Supabase 우선 / 로컬 JSON fallback (`pilgiLoader.ts`)
- JSON 파일은 Vite lazy load chunk로 분리 (초기 로드 영향 없음)

### 주요 파일

| 파일 | 설명 |
|------|------|
| `src/data/pilgiQuestions.json` | 3,900문항 데이터 (2.45MB) |
| `src/data/pilgiLoader.ts` | Supabase/로컬 데이터 로더 |
| `scripts/parse-pdf.cjs` | PDF → JSON 파싱 스크립트 |
| `scripts/parse-explanations.cjs` | 해설집 → explanation 병합 |
| `sql/insert-pilgi-questions.sql` | Supabase INSERT 스크립트 (704KB) |

## 3. 버그 수정

### 필기시험 시간 (150분 → 100분)

6곳에서 필기시험 시간이 150분으로 잘못 표시. 100분으로 수정.
- `site.js`, `ExamInfo.tsx`, `PracticeHome.tsx`, `PilgiHome.tsx`, `ExamSelect.tsx`, `Home.tsx`

### 필기시험 레이아웃 (150px → 전체 폭)

`.exam-container`의 2컬럼 그리드가 `.exam-body`가 아닌 전체 컨테이너에 적용되어
문제 카드가 280px 컬럼에 갇힘. 그리드를 `.exam-body`로 이동하여 수정.

### 푸터 하단 고정

콘텐츠 적은 페이지에서 푸터 떠 있는 문제. `app-layout` flex 컨테이너 + `min-height: 100vh` 적용.
